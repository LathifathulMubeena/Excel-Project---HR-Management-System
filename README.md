## 👥 HR Management System
An end-to-end Excel-based HR system with 7 interconnected modules for tracking 
the full employee lifecycle.

**Modules:**
- 📊 Dashboard — real-time HR metrics overview
- 👥 Employee Data — centralized employee records
- 📋 Recruitment — hiring pipeline tracking
- ⭐ Performance — employee performance evaluation
- 💰 Payroll — salary & compensation tracking
- 📈 Forecast — workforce planning projections
- 🎓 Training — employee development tracking

**Highlights:**
- Built dynamic dashboard using VLOOKUP, COUNTIFS, AVERAGEIF & IFERROR
- Applied conditional formatting for at-a-glance status tracking
- Designed 3 interactive charts for HR KPI visualization

**Skills:** Excel, Advanced Formulas, Dashboard Design, HR Analytics, Data Visualization
